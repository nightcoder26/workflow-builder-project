# Workflow Automation Platform (Frontend)

A no-code visual workflow builder using React, React Flow, Tailwind CSS, and Zustand.

- Left sidebar: connections + node palette
- Main canvas: React Flow with draggable nodes and connections
- Top bar: Save, Test, History, Profile

## Quick Start

1. Install dependencies
2. Run dev server

## Notes

- OAuth flows are mocked in the frontend for demo purpose. Replace with real API endpoints.
- Sample workflow is pre-populated.
